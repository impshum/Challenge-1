STEP 2

  • Open up phpmyadmin at http://localhost/phpmyadmin.
  • Create a database.
  • Create a table inside this database with 4 columns called ID, username, email and password.
    • The ID column needs to assign a unique number to each entry (auto increment).
    • The other columns are variable characters at whatever length you seem suitable (anything up to 255 will do).
    • The primary key needs to be ID.
      +-----+------------+-------------+---------------+
| ID  |  username  |    email    |    password   |
+-----+------------+-------------+---------------+
|  1  |     a      |      @      |       *       |
|  2  |     b      |      @      |       *       |
|  3  |     c      |      @      |       *       |
    • Try researching examples online first but if you get really frustrated a template is here.
    • When you're done take a break. Click HERE.

$connect = mysql_connect("localhost", "root", "root"); // Database login details - ("location", "user", "pass")

if (!$connect) {
    die('Could not connect: ' . mysql_error());
}

mysql_select_db("___", $connect); // enter the name of your database

$data="INSERT INTO ___ (username, email, password) VALUES ('$_POST[username]','$_POST[email]','$_POST[password]')"; // enter the name of your table

if (!mysql_query($data, $connect)) {
    die('Error: ' . mysql_error());
}

mysql_close($connect); // Run for the hills

echo "Record saved to database";
